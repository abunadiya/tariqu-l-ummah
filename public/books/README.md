# Каталог PDF-книг

Поместите файлы в эту папку и добавьте запись в:
1. `src/pages/Library.jsx` → массив `LIBRARY_SECTIONS`
2. `src/locales/ru.json` и `fr.json` → `library.books.<id>`

Категории библиотеки согласованы с `.cursorrules`:
- Хадис и сира
- Исламская география и первоисточники
- Всемирная история Уммы / Халифат
- Поволжье и Восточная Европа
- Интеллектуальное наследие Кавказа
- Лингвистика и письменность

Не включайте пустые/битые PDF (например, одностраничные без текста).
При аннотациях соблюдайте акыду Ахлю-с-сунна и салават ﷺ.
